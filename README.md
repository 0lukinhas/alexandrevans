## 🌐 Sobre o Projeto

Site institucional da **Alexandre Vans**, empresa especializada em aluguel de vans com motorista na região de Itatiba/SP. O site foi desenvolvido com foco em:

- **SEO Local** — Schema JSON-LD, sitemap, robots.txt, llms.txt para indexação por IAs
- **Performance** — HTML/CSS/JS puro sem dependências pesadas
- **Design Premium** — Dark mode, glassmorphism, animações suaves
- **Conversão** — Formulário integrado ao WhatsApp, FAQ interativo, depoimentos

---

## 📁 Estrutura do Projeto

```
alexandrevans/
```

## 🚀 Como Executar Localmente

Basta abrir o `index.html` em qualquer navegador moderno — sem necessidade de servidor ou build.

```bash
# Opcionalmente, use um servidor local simples:
npx serve .
# ou
python -m http.server 8080
```

## 📡 SEO e Indexação

| Arquivo | Finalidade |
|---|---|
| `robots.txt` | Autoriza todos os crawlers |
| `sitemap.xml` | Entrega ao Google Search Console |
| `llms.txt` | Indexação por assistentes de IA |
| Schema JSON-LD | Aparece no Knowledge Graph do Google |

**Próximos passos recomendados:**
1. Registrar domínio `alexandrevans.com.br`
2. Criar perfil no **Google Meu Negócio**
3. Enviar `sitemap.xml` ao **Google Search Console**
4. Substituir imagens de IA pelas fotos reais da frota
